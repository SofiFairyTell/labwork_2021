Чтобы решить задачу брендинга, чтобы повысить осведомленность об игре, повысить создание спроса будем использовать следующие инструменты:
Медийная реклама(телевизор) радио. Печатная реклама (журналы, газеты), SMM, Тизерная реклама, PR публикации.

Для повышение доверия, подкрепление выбора игры, повышение лояльности потребителей используем:  SMM, PR публикации

Для еще большего увеличения кол-ва поьзователей: контекстная реклама, Таргетированная реклама, SEO
